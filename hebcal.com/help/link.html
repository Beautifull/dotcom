Linking from your Synagogue to Hebcal

Can my synagogue link to hebcal.com?

Yes, you can link to hebcal.com. You can also add some hebcal.com content directly to your website using the following instructions.

How can my synagogue put 1-Click Shabbat candle-lighting times on its own website?

You can use the Link to 1-Click Shabbat page to generate HTML tags to insert weekly candle-lighting times directly on your web page.

How can my synagogue display a Jewish Calendar on its own web site?

You can use the HTML tags below, courtesy of Kevin Ilsen' JavaScript Event Calendar to put a calendar on your website.

<div style="font-family: Verdana,Arial,Helvetica">
<script type="text/javascript" language="JavaScript"
 src="http://www.hebcal.com/i/calendar.js">
</script>
<script type="text/javascript" language="JavaScript"
 src="http://www.hebcal.com/hebcal/?v=1;cfg=e;nh=on;nx=on;year=now;month=x">
</script>
<script language="JavaScript">
 <!--
  SpecialDay=7;
  FontSize="+1";
  Calendar();
 // -->
</script>
</div>

To add candle-lighting times for your town, you can add ;c=on;zip=90210;m=72;s=on to the end of the 2nd URL (be sure to change 90210 to your shul's zip code). For example, 

<script type="text/javascript" language="JavaScript"
 src="http://www.hebcal.com/hebcal/?v=1;cfg=e;nh=on;nx=on;year=now;month=x;c=on;zip=90210;m=72;s=on">
</script>

How can my synagogue link to the Yahrzeit, Birthday and Anniversary Calendar from its own website?

You can use these HTML tags to insert a small form directly on your web page, with a reverse link back to your site:

<form action="http://www.hebcal.com/yahrzeit/">
<input type="hidden" name="ref_url" value="http://www.tbala.org">
<input type="hidden" name="ref_text" value="Temple Beth Am">
<select name="t1">
<option value="Yahrzeit">Yahrzeit</option>
<option value="Birthday">Birthday</option>
<option value="Anniversary">Anniversary</option>
</select>

Month:
<select name="m1" id="m1">
<option value="1">January</option>
<option value="2">February</option>
<option value="3">March</option>
<option value="4">April</option>
<option value="5">May</option>
<option value="6">June</option>
<option value="7">July</option>
<option value="8">August</option>
<option value="9">September</option>
<option value="10">October</option>
<option value="11">November</option>
<option value="12">December</option>
</select>

Day:
<input type="text" name="d1" size="2" maxlength="2" id="d1">
Year:
<input type="text" name="y1" size="4" maxlength="4" id="y1">
&nbsp;&nbsp;&nbsp;
Name:
<input type="text" name="n1" id="n1"><br>

<input type="checkbox" name="yizkor" id="yizkor">
<label for="yizkor">Include Yizkor dates</label><br>
<input type="submit" value="Compute Calendar">
</form>

Make sure you replace both the ref_url and ref_text value fields with with your own synagogue's.
