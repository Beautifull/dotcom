What is the Hebcal Interactive Jewish Calendar?

Hebcal Interactive Jewish Calendar is a web site that lets you generate a list of Jewish holidays for any year. Candle lighting times are calculated from your latitude and longitude (which can be determined by your zip code or closest city).

Who maintains this service?

Michael J. Radwin created and runs the web interface. Danny Sadinoff wrote hebcal 3.2 for UNIX, the back-end that this service uses to generate the list of Holidays.

What's different about this web interface?

This site offers:

  • Candle-lighting times customized to one of 43,000 zip codes (instead of just a list of 34 cities)
  • Download calendar in Outlook CSV format
  • Download calendar in Palm Date Book Archive (.DBA)
  • Add events to Yahoo! Calendar
  • Totally free; no ads, no fees -- all you need is a web browser
  • Both transliterations and actual Hebrew text with vowels

Specifics about Jewish Holidays

When do the Holidays begin?

All Jewish Holidays begin the evening before the date specified. This is because the Jewish day actually begins at sundown on the previous night. Sometimes, for clarity, the Erev Holiday is also included.

When does Shabbat begin?

Shabbat begins 18 minutes before sundown on Friday night. In Jerusalem, Shabbat begins 40 minutes before sundown.

How do I import Hebcal entries into my PDA or desktop calendar program?

Outlook CSV

  • Open Microsoft Outlook
  • Select the File / Import and Export... Menu option
  • Select "Import from another program or file" and click Next
  • Select "Comma Separated Values (Windows)" and click Next
  • Find the hebcal.csv file you downloaded and select "Do not import duplicate items" and click Next.
  • Selct Calendar and click Next. This will import your chosen file.

Palm Date Book Archive (DBA)

  • Open Palm Desktop for Windows version 2.0 or greater
  • Go to the Date Book view
  • Select the File / Import menu option
  • Find the hebcal.dba file and choose Open. This will import the chosen file.

How are candle lighting times determined from my zip code?

Since candle lighting times are determined from latitude and longitude, a process called geocoding is used to determine the geographic position of a zip code. Your zip code is looked up in a geocoded database of about 43,000 zip codes.

Two-thirds of the zip codes were downloaded from The U.S. Census Bureau's Gazetteer project, based on 1990 Census data. The other third of the zip codes that aren't listed in the Gazetteer database were obtained from an Internet mapping service (available via extra-zips.txt.gz, 161 Kbytes, in the same format as the Gazetteer database). Also usps-citynames.txt.gz, 188 Kbytes, contains an updated mapping from zip codes to city names obtained from the US Postal Service.

Who's using this site to get candle-lighting times?

Check out top-zips.txt, a list of all of the zip codes people have used (updated daily).

What are other good Jewish calendar resources on the web?

For easy access to weekly Shabbat times, you might try our own 1-Click Shabbat page.

Judaism 101's Jewish Calendar is an excellent overview. Also, you might see Yahoo!'s Judaism Resources: Calendars page. If you're just looking for a quick table of major Jewish Holidays, I'd suggest B'nai B'rith's 7-year Calendar of Jewish Holidays.

Chabbad.org has a configurable monthly calendar which also uses zip codes to determine candle-lighting times. Their zip code database is smaller than hebcal.com, but they also compute sunrise and sunset for every day of the week (instead of just Shabbat and Havdalah).

If you're looking for more than just candle-lighting times, check out Kashrut.com's Zemanim (prayer time) Service. They can compute Alos HaShachar, Sunrise, Sunset, Sof Zeman K'rea Sh'ma, Plag HaMincha, Tzait hakochavim, etc.

How can my synagogue put 1-Click Shabbat candle-lighting times on its own website?

You can use these HTML tags to insert weekly candle-lighting times directly on your web page:

<script type="text/javascript" language="JavaScript"
src="http://www.hebcal.com/shabbat/?zip=90210&cfg=j">
</script>
<noscript>
<a href="http://www.hebcal.com/shabbat/?zip=90210">1-Click
Shabbat Candle Lighting times</a>.
</noscript>

Make sure you replace both zip codes with your synagogue's zip code.

What's the Hebcal Cookie?

A cookie is a small message given from our web server to your browser that stores website preferences. The Hebcal Cookie stores information like your zip code and time zone, and also preferences such as whether you want Ashkenazis Hebrew or the weekly Torah portion to be listed in your Hebcal results.

To delete your Hebcal Cookie at any time, click the Delete my cookie link.

What does CH''M mean?

CH''M is an abbreviation for Chol Ha-Mo'ed. Chol Ha-Mo'ed are the intermediate days of Passover and Sukkot, when work is permitted.

Can you recommend any books on the Hebrew Calendar or Jewish Holidays?

For a good reference book on the major Jewish Holidays, I'd suggest Rabbi Michael Strassfeld's The Jewish Holidays: A Guide & Commentary.

Rabbi Nathan Bushwick's Understanding the Jewish Calendar explains some of the mathematical and scientific underpinnings of the calendar.
