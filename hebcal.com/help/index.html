About the Hebcal Interactive Jewish Calendar

This page describes the Hebcal Interactive Jewish Calendar, available at http://www.radwin.org/hebcal/.

1. What is the Hebcal Interactive Jewish Calendar?

Hebcal Interactive Jewish Calendar is a web site that lets you generate a list of Jewish holidays for any year. Candle lighting times are calculated from your latitude and longitude (which can be determined by your zip code or closest city).

2. Who maintains this service?

Michael J. Radwin created and runs the web interface. Danny Sadinoff wrote the hebcal 3.2, the unix back-end that this service uses.

3. How are candle lighting times determined from my zip code?

Since candle lighting times are determined from latitude and longitude, a process called geocoding is used to determine the geographic position of a zip code. Your zip code is looked up in a geocoded database of about 43,000 zip codes.

Two-thirds of the zip codes were downloaded from The U.S. Census Bureau's Gazetteer project, based on 1990 Census data. The other third of the zip codes that aren't listed in the Gazetteer database were obtained from an Internet mapping service (available via extra-zips.txt.gz, 161 Kbytes, in the same format as the Gazetteer database).

4. What are popular zip codes people have been using for candle-lighting times?

Check out top-zips.txt, updated daily.

6. What are other good Jewish calendar resources on the web?

The Judaism and Jewish Resources page has a few good links to other sites. Also, you might see Yahoo! Society and Culture:Religion and Spirituality:Faiths and Practices:Judaism:Resources:Calendars. If you're just looking for a quick table of major Jewish Holidays, I'd suggest B'nai B'rith's 7-year Calendar of Jewish Holidays.

If you're looking for more than just candle-lighting times, check out Kashrut.com's Zemanim (prayer time) Service. They can compute Alos HaShachar, Sunrise, Sunset, Sof Zeman K'rea Sh'ma, Plag HaMincha, Tzait hakochavim, etc.

7. I'm a programmer. Can I see the source code?

  • zips2db.pl, perl script that translates a Gazetteer zip code CSV file into a Berkeley DBM hash file.
