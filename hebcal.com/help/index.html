About the Hebcal Interactive Jewish Calendar

This page describes the Hebcal Interactive Jewish Calendar, available at http://www.radwin.org/hebcal/.

1. What is the Hebcal Interactive Jewish Calendar?

Hebcal Interactive Jewish Calendar is a web site that lets you generate a list of Jewish holidays for any year. Candle lighting times are calculated from your latitude and longitude (which can be determined by your zip code or closest city).

2. Who maintains this service?

Michael J. Radwin created and runs the web interface. Danny Sadinoff wrote the hebcal 3.2, the unix back-end that this service uses.

3. How are candle lighting times determined from my zip code?

Geographic zip code information was downloaded from The U.S. Census Bureau's Gazetteer. Their database maps zip codes to latitude and longitude, and candle lighting times are determined from that information.

Additionally, a handful of zip codes that aren't listed in the Gazetteer database were obtained from an Internet mapping service. I've created extra-zips.txt in the same format as the Gazetteer database.

4. What are popular zip codes people have been using for candle-lighting times?

Check out top-zips.txt, updated daily.

The Judaism and Jewish Resources page has a few good links to other sites. Also, you might see Yahoo! Society and Culture:Religion and Spirituality:Faiths and Practices:Judaism:Resources:Calendars.

7. I'm a programmer. Can I see the source code?

Sure thing:
  • hebcal.pl, source code to the CGI form.
  • ChangeLog.txt, revision history of hebcal.pl.
  • zips2db.pl, perl script that translates a Gazetteer zip code CSV file into a Berkeley DBM hash file.
  • hebcalzips.pl, perl script that reports which are the top zip codes people have been using for candle-lighting.
